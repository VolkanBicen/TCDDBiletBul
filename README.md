# TCDDBiletBul
TCDD  hızlı Tren bilet arama ve telegram'dan uyarı verme botu

https://ebilet.tcddtasimacilik.gov.tr/
Adresinden bilet arama işlemi yapan Selenium ve TCDD-Bilet telegram botu üzerinden mesaj gönderen, bilet arama ve uyarı verme uygulamasıdır.
